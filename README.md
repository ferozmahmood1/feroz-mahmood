# Feroz Mahmood Portfolio

Static, Vercel-ready personal portfolio for Feroz Mahmood.

## Contents

- `index.html` - portfolio markup
- `styles.css` - responsive visual design
- `script.js` - reveal animations, active nav state, and mailto contact form
- `assets/feroz-headshot.jpg` - optimized headshot
- `assets/Feroz_Mahmood_Resume.pdf` - downloadable resume
- `vercel.json` - Vercel static deployment settings

## Deploy To Vercel

1. Create a GitHub repository named `feroz-mahmood`.
2. Upload these files to the repository root.
3. In Vercel, choose `Add New Project`.
4. Import the GitHub repository.
5. Leave framework preset as `Other`.
6. Deploy.

The site does not need a backend or environment variables.
